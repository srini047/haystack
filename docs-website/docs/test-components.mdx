## Code Blocks

### Shell (single-line)

```shell
pip install haystack-ai
```

### Python (multi-line)

```python
from haystack import Pipeline, Document
from haystack.document_stores.in_memory import InMemoryDocumentStore

document_store = InMemoryDocumentStore()
document_store.write_documents(
    [
        Document(content="My name is Jean and I live in Paris."),
        Document(content="My name is Mark and I live in Berlin."),
    ],
)

pipeline = Pipeline()
print(pipeline)
```

### YAML

```yaml
components:
  retriever:
    type: haystack.components.retrievers.InMemoryBM25Retriever
    init_parameters:
      document_store: document_store
pipelines:
  query:
    - retriever
```

## Tables

### Standard Table

| Component | Type | Description |
| --- | --- | --- |
| `InMemoryDocumentStore` | Document Store | Stores documents in memory for fast prototyping. |
| `OpenAIChatGenerator` | Generator | Generates chat completions using OpenAI models. |
| `InMemoryBM25Retriever` | Retriever | Retrieves documents with BM25 ranking. |
